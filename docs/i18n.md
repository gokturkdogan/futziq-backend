---

## i18n bundle

```http
GET /api/v1/meta/i18n-bundle?locale=tr
Accept-Language: tr
```

**Örnek yanıt:**

```json
{
  "errors": {
    "errors.NOT_YOUR_TURN": "Sıra sende değil",
    "errors.PLAYER_NOT_ELIGIBLE": "Bu oyuncu bu pozisyon için uygun değil"
  },
  "enums": {
    "enums.performance.PERFECT": "Mükemmel",
    "enums.performance.EXCELLENT": "Harika"
  },
  "slots": {
    "slots.GK": "Kaleci",
    "slots.DEF1": "Defans 1",
    "slots.ATT": "Forvet"
  }
}
```

**Flutter:** Bundle'ı memory'de cache'leyin; locale değişince yeniden çekin.

```dart
final bundle = await client.get<Map<String, dynamic>>('/api/v1/meta/i18n-bundle');
final errors = bundle['errors'] as Map<String, dynamic>;
final text = errors['errors.NOT_YOUR_TURN'] ?? 'Unknown error';
```

## Enum çevirileri

```http
GET /api/v1/meta/enums?locale=tr
```

`performanceRating`, `objective` gibi enum değerlerinin çevirileri.

---

## Lineup slot displayName

Session ve result yanıtlarındaki `lineup[].displayName` alanı `Accept-Language` header'ına göre backend tarafından resolve edilir. Client tarafında ek çeviri gerekmez; fallback olarak `slots.{slotCode}` bundle key'ini kullanın.

## Katalog lokalizasyonu

`GET /api/v1/game-families` ve `GET /api/v1/game-families/:code` yanıtlarındaki `title`, `description` alanları `Accept-Language` ile lokalize edilir.

**Cache key:** `{familyCode}:{locale}:{catalogVersion}`
